## 📈 Project Overview
Bitcoin is a highly volatile cryptocurrency. Predicting its price movement is critical for investors, traders, and policymakers to mitigate risk and optimize returns. Due to the high noise and non-linear patterns in financial time-series data, traditional forecasting methods are highly inaccurate. 

This project designs, trains, and evaluates a machine learning regression model to forecast the Close price of Bitcoin based on historical market indicators and time-series feature engineering.

### 🛠️ Technology Stack
- **IBM Cloud & watsonx.ai Studio:** Cloud platform hosting, model training, and online deployment runtime.
- **AutoAI / IBM Granite:** For automated feature engineering, pipeline optimization, and algorithm comparison.
- **Jupyter Notebook (Python 3.10):** Offline model replication, data scaling, and visual evaluation.
- **Scikit-Learn & PyMuPDF:** For model training, regression metrics calculation, and report generation.
